ASP.NET CORE demo for JSNLog

  • The JavaScript for this demo is in the home component. It uses JSNLog to create a log message and send it to the server.
  • On the server, this demo sends all log entries to the output window.
  • Install JSNLog in your own ASP.NET CORE application.
Log JavaScript exception

Calls JavaScript function GenerateException (defined in the home component) that causes an exception by trying to access a variable that doesn't exist. JSNLog will handle this uncaught exception and send it to the server, where it will be logged.


Log message from JavaScript

Calls JavaScript function LogMessage (defined in the home component) that logs a message. JSNLog will send the message to the server where it will be logged.
